# python-if_else_loops_functions
This project contains python exercises focused on conditional  statements, loops, and functions as part of the ALU Higher level programming track.

## Topic Covered
-if/elif/else conditional statements
-while and for loops
-Functions and their prototypes
- Input validation and number manipulation

## Task 
- 0-positive_or_negative.py: Prints whether a random number is positive, negative, or zero
- 1-last_digit.py: Prints and evaluates the last digit of a random number
- 2-print_alphabet.py: Prints the lowercase ASCII alphabet using a loop
